**公開サイト:** [https://amekusa.github.io/shiftVet/](https://amekusa.github.io/shiftVet/)

このプロジェクトは、Webサイトのローカルアーカイブ（ミラー）です。
HTTrack Website Copierによって生成されたコンテンツを含みます。

## ディレクトリ構成

- `index.html`: プロジェクトのメインインデックスページ
- `shiftVet/`: アーカイブされたサイトデータが含まれるディレクトリ

## 開発環境

推奨されるエディタ設定は `.editorconfig` に記述されています。

## ライセンス

コンテンツの著作権は元のサイト所有者に帰属します。
